# REST API Testing -- User Management System

## Project Overview

This project demonstrates REST API testing using Postman. The APIs
belong to a User Management system and support CRUD operations.

## Tools Used

-   Postman
-   JSON
-   Excel
-   GitHub

## API Endpoints Tested

-   GET /api/users?page=2
-   POST /api/users
-   PUT /api/users/2
-   DELETE /api/users/2

## Testing Scope

-   Status code validation
-   JSON response validation
-   CRUD operation testing
-   API reliability validation

## How to Run

1.  Import the Postman collection into Postman.
2.  Run each API request.
3.  Validate response status codes and JSON responses.
4.  Record test results in the Excel test case file.
